## Requirements

| Target            | Version |
|-------------------|---------|
| iOS               |  => 13.0 |
| macOS         |  => 10.15 |
| watchOS      |  => 5.0 |
| tvOS             |  => 13.0 |
| Swift             |  => 5.3 |

## Installation
### Use with Swift Package Manager:

  Add the following line to your `Package.swift`:

  ```swift
 dependencies: [
	 // ...
	 .package(url: "https://https://github.com/mcostasilva/bitkub-swift.git"),
 ],
 ```
